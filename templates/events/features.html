Features

Everything built into Community Playlist — submitted by the community, for the community.

v0.1 — Portland, OR
Discover events
≡

List, Map & Hybrid views

Browse upcoming events as a calm scrollable list, explore by map for discovery, or use hybrid mode for side-by-side research. Your preference is saved automatically.

⚙

Smart filters

Filter by genre, event type (Music / Bike / Fundraiser / Food / Hybrid), Online vs Local, neighborhood, and date range (Today / This Week / This Month / Any Future / All Past).

🗺

Interactive map

Hover a list card to highlight its map marker. Click a marker to scroll the list to that event. Flyer thumbnails appear in map popups.

📅

Calendar subscribe

Subscribe to a live iCal feed in Google Calendar, Apple Calendar, or any calendar app. Build a custom filtered feed →

Event pages
🌹

The Flyer

The first photo uploaded to an event is featured as "The Flyer" — full-width, orange-bordered, and labeled. Additional photos form a gallery below.

🔍

Photo lightbox

Click any photo to view it full-screen. Close with Escape or a click outside the image.

📍

Smart location links

Addresses link to OpenStreetMap directions. URLs entered as the location link directly to the website. Calendar icon adds the event to Google Calendar with one tap.

📸

Community photo uploads

Anyone can add photos to an event after it happens. Captions and submitter credit supported.

Submit & manage
✦

Submit an event

Address auto-geocodes as you type with a live map preview. Neighborhood is auto-detected. Genre tags, multiple links, event type, flyer photo — all supported.

🔑

Claim your event New

Register with the email you used to submit an event and it's automatically claimed. Edit details, track approval status, and manage from your dashboard.

🎟

Event categories

Music, Bike, Fundraiser, Food, and Hybrid — color-coded badges on every card and event page. Filter the whole list by category.

Community
💬

Discord community

Ideas, feedback, and ticket suggestions live in #website-communications. React 🌹 to vote for your favorites.

🌹

Visitor counter

A quiet count of unique visitors who've passed through — one per session, no tracking, no ads, no data sold.

Open data & community support

This project runs on open data and community-built tools. No ads, no tracking, no investors — just people sharing what's happening. These are the shoulders we stand on. Please consider supporting them.

🎵

ArtistData Support them

ArtistData helps musicians manage and sync their tour dates across the web — keeping live music data open and accessible. We use their data to help surface Portland shows. Visit ArtistData →

🎸

MusicBrainz

The open, community-maintained music encyclopedia. We pull concert data from MusicBrainz to help populate Portland events. Free, nonprofit, no strings attached. Donate to MetaBrainz →

📅

Calagator

Portland's own open-source community calendar. Tech meetups, civic events, workshops — Calagator has been covering PDX since 2008. We import their events automatically. calagator.org →

🗺

OpenStreetMap

All maps and geocoding on this site use OpenStreetMap — the community-built, freely editable map of the world. No Google, no tracking. About OSM →
