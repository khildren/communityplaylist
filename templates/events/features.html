Discover events
≡

List, Map & Hybrid views

Browse upcoming events as a calm scrollable list, explore by map for discovery, or use hybrid mode for side-by-side research.

🔍

Search & smart filters

Full-text search across title, description, and location. Filter by genre, category (Music / Arts / Bike / Fundraiser / Food / Hybrid), neighborhood, date range, and free-only.

🗺

Interactive map

Hover a list card to highlight its marker. Click a marker to scroll to that event. Flyer thumbnails in popups. All powered by OpenStreetMap — no Google, no tracking.

📅

Calendar subscribe

Subscribe to a live iCal feed in Google Calendar, Apple Calendar, or any calendar app. Build a custom filtered feed →

Submit & manage
✦

Submit an event

Address auto-geocodes with a live map preview. Neighborhood is auto-detected. Genre tags, artist tags (auto-detected from the event title), multiple links, flyer photo — all supported.

🎤

Artist tagging New

Tag performing artists on any event. Type a name and it searches our database, then MusicBrainz if not found. Artists are auto-detected from event titles (e.g. "ft.", "w/", "presents").

🔁

Recurring events New

Mark an event as recurring — weekly, monthly, every 3rd Thursday, annually, and more. Future instances generate automatically. Resident artists are tagged on every instance.

🔑

Claim your event

Register with the email you used to submit and your events are automatically claimed. Edit details and track approval from your dashboard.

Event pages
🌹

The Flyer

The first photo uploaded is featured as the flyer — full-width, orange-bordered. Additional photos form a gallery.

📍

Smart location links

Addresses link to OpenStreetMap directions. Calendar icon adds the event to Google Calendar with one tap.

📸

Community photo uploads

Anyone can add recap photos after an event happens. Captions and submitter credit supported.

The PDX event net

Community Playlist automatically pulls events from across Portland so nothing slips through the cracks. All imports are geocoded, categorized, and genre-tagged automatically.

📡

Venue iCal feeds

Dozens of PDX venues, collectives, and promoters publish iCal feeds. We import them twice a week and auto-detect new ones monthly.

🎸

MusicBrainz

The open, community-maintained music encyclopedia. We pull Portland concert data and use it for artist search and tagging. Free, nonprofit. Donate to MetaBrainz →

📅

Calagator

Portland's own open-source community calendar, covering tech meetups, civic events, and workshops since 2008. We import their events automatically. calagator.org →

🗺

OpenStreetMap & Nominatim

All maps and geocoding run on OpenStreetMap — community-built, freely editable, no tracking. About OSM →
