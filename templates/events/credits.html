Open Source Love

Community Playlist is built on the shoulders of giants — open source projects, open data services, and community-maintained infrastructure. This page is our thank-you note.

Core Framework
🐍

Django Python

The web framework that powers everything — models, views, migrations, auth, admin, and more. Community Playlist runs on Django 4.2. djangoproject.com →

🚀

Gunicorn WSGI

Python WSGI HTTP server — handles the traffic in production. gunicorn.org →

🐘

PostgreSQL

Open source relational database — stores every event, artist, venue, profile, and track on the platform. postgresql.org →

📦

Docker + Unraid

Community Playlist runs in a Docker container on a self-hosted Unraid NAS in Portland, OR. Zero cloud, zero tracking, zero AWS bills. docker.com →

Maps & Geo
🗺

Leaflet.js

The lightweight, mobile-friendly interactive map library powering all map views — event markers, venue pins, autopilot routes, and neighborhood overlays. leafletjs.com →

🌍

OpenStreetMap & Nominatim

Community-built map data and geocoding. Every event address is geocoded via OSM Nominatim — no Google Maps, no tracking, fully open. About OSM → · Nominatim →

🛣

OSRM (Open Source Routing Machine)

Driving route computation for the map autopilot feature — real road routes between venues, no Google Directions API needed. project-osrm.org →

Music & Data Services
🎸

MusicBrainz / MetaBrainz

Open music encyclopedia — artist search, tagging, and Portland concert data imports. 100% community-maintained, no ads, no paywall. musicbrainz.org → · Donate →

🎧

ListenBrainz

Open scrobbling platform from MetaBrainz. Your recent listens display live on your profile and refresh every 30 seconds — fully open API, no key required, no tracking. listenbrainz.org →

〜

Last.fm

Music scrobbling and listening history. Your top tracks of the month appear as a playable card on your public profile. last.fm →

💿

Discogs

The world's largest music marketplace and database. Your public vinyl collection shows with cover art on your profile — click any record to search and play it via YouTube. discogs.com →

▶

YouTube Data API

Powers the YouTube channel cards on profiles — playlists, video grids, and search-to-play. Quota-cached to stay well under daily limits. YouTube API docs →

Event Data Sources
🎛

19hz.info — PNW Electronic Music Calendar

The original Pacific Northwest rave and electronic music calendar, running since 1997. Oregon shows are imported automatically — raves, warehouse parties, club nights that never make Ticketmaster. 19hz.info →

🎚

house-mixes.com

Community platform for DJ mixes and sets. Artists can link their house-mixes.com profile to stream mixes directly on their Community Playlist artist page — playable in the CP player, no redirects. Thanks to Encrypted for bringing this one to the community. house-mixes.com →

📅

Calagator

Portland's open-source community calendar since 2008 — tech meetups, civic events, workshops. calagator.org →

📡

iCal / iCalendar (RFC 5545)

The open standard for calendar data. Dozens of PDX venues publish iCal feeds — we import them all via the icalendar Python library. icalendar on PyPI →

Frontend & Other Libraries
🌤

Open-Meteo

Free, open-source weather API — no key, no tracking. Powers real-time weather and PM2.5 air quality in the site header. open-meteo.com →

📆

rrule.js

JavaScript library for recurring event rule parsing — powers "every 3rd Thursday", "weekly on Fridays", and all other recurrence patterns in the calendar. github.com/jakubroztocil/rrule →

🔗

Traefik

Reverse proxy and SSL termination — handles HTTPS, routing, and Let's Encrypt certificate renewal automatically. traefik.io →

⬡

Bandcamp, SoundCloud, Mixcloud, Spotify, Apple Music

Embedded music players via official embed iframes — no tracking beyond what each platform's embed provides. Playable inline on artist and user profiles.

🌹 Community Playlist is open source

All code is public on GitHub. File bugs, suggest features, or fork the whole thing. Built by and for the Portland community — no ads, no tracking, no investors, no corporate agenda.
